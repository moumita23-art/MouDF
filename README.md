# 📄 VSCode PDF Replicator

A curated workflow for creating pixel-perfect PDF replicas directly inside Visual Studio Code. This repository includes configurations, templates, and automation scripts to turn Markdown into perfectly formatted PDFs.

## 🚀 Features
- **Instant Conversion:** Convert Markdown to PDF within VS Code.
- **Custom Styling:** Use CSS to create custom PDF replicas.
- **Automation:** Streamline workflows using Markdown PDF.
- **PDF Manipulation:** (Optional) Include features like split/merge.

## 🛠️ Setup
1. Open Visual Studio Code.
2. Install the [Markdown PDF](https://github.com/yzane/vscode-markdown-pdf) extension.
3. Use `Ctrl+Shift+P` and select `Markdown PDF: Export (pdf)`.

## 📝 Usage
- **Style:** Create a `style.css` file to define your replica's layout.
- **Convert:** Open your `.md` file, right-click, and select "Markdown PDF".
